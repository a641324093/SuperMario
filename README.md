## 开发平台
Java + Swing控件  
Eclipse IDE

## 游戏内容
 - 界面，多层背景横轴滚动，菜单与游戏界面的切换。
 - 算法，模拟实现重力与物理碰撞等效果。
 - 内容，实现了长大蘑菇、？号宝箱、栗子怪、食人花、乌龟等多种元素。
 - 音效：背景音乐的切换，动作音效的实现。

## 相关技术点
 - Swing实现基本窗体与部分组件
 - 双缓冲解决卡帧现象
 - 逻辑、 音乐、 绘图、 音效多线程控制
 - 算法模拟重力系统以及多块物理碰撞
 - 帧动画的应用， 素材的组合与绘制
 - javazoom 实现背景音乐播放
 - AudioData 实现音效处理。
